# Automatisierte-Webentwicklung-durch-KI: Eine Analyse der Barrierefreiheit und des Einflusses von Prompting
Dieses Repository enthält die Zusatzmaterialien des wissenschaftlichen Artikels *Automatisierte-Webentwicklung-durch-KI: Eine Analyse der Barrierefreiheit und des Einflusses von Prompting* aus dem Modul Wissenschaftliches Arbeiten des Wintersemesters 2025/2026. Es wurde untersucht, inwieweit ChatGPT GPT-5 und Gemini 2.5 Pro automatisch erkennbar barrierefreien Webseitencode generieren und welchen Einfluss ein Barrierefreiheitshinweis hat.

## Prompts (Aufgabenstellungen)
Hier sind die Aufgaben, die den Modellen als Prompts gestellt werden, zu finden: 
- [Prompts](https://github.com/majbrit/Automatisierte-Webentwicklung-durch-KI/tree/main/Prompts)

## Generierter Code (Output)
Der Generierte Code ist nach Aufgabe und Promptvariante in Ordner unterteilt. Sie enhalten den vollständigen HTML-, CSS- und JavaScript-Code.

*Hinweis:* Der Begriff "Barrierefrei" im Ordnernamen bezieht sich auf den **Prompt**, nicht zwingend auf die Qualität des Codes.
- [Code_ChatGPT](https://github.com/majbrit/Automatisierte-Webentwicklung-durch-KI/tree/main/Code_ChatGPT)
- [Code_Gemini](https://github.com/majbrit/Automatisierte-Webentwicklung-durch-KI/tree/main/Code_Gemini)

Der Code zur Durchführung des Wilcoxon-Vorzeichen-Rang-Test und der Effektstärke sowie die Ausgabe dazu ist hier zu finden:
